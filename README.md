# 📖 Repositório - Entendendo Algoritmos

Este repositório contém a resolução de exercícios e implementações baseadas no livro *Entendendo Algoritmos - Um Guia Ilustrado para Programadores e Outros Curiosos*, de **Aditya Bhargava**.

## 📌 Sobre o Livro
O livro apresenta os principais algoritmos da computação de forma visual e acessível, sendo uma excelente introdução para programadores iniciantes e também um reforço para quem deseja aprofundar seus conhecimentos sobre eficiência e estrutura de dados.

Dentre os temas abordados, temos:
- Busca Binária
- Complexidade de Algoritmos (Big O)
- Estruturas de Dados (Listas, Pilhas, Filas, Hash Tables)
- Algoritmos de Ordenação (QuickSort, MergeSort)
- Programação Dinâmica
- Grafos e Algoritmos de Caminho Mínimo (Dijkstra)

## 📂 Estrutura do Repositório
```
📁 entendendo-algoritmos
│── 📄 README.md       # Visão geral do repositório
│── 📂 busca-binaria   # Implementações e exercícios sobre busca binária
│── 📂 ordenacao       # Algoritmos de ordenação
│── 📂 grafos          # Implementações envolvendo grafos
│── 📂 estrutura-dados # Listas, filas, pilhas, tabelas hash
```

## 🚀 Objetivo
Este repositório servirá como uma aplicação dos conceitos abordados.
